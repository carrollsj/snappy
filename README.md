# Snappy

Snappy is a macOS screenshot and markup tool written in Python.

## Features

- Quick region capture to clipboard with `Cmd+Shift+1`
- Region capture with markup editor using `Cmd+Shift+2`
- Simple drawing and annotation workflow

## Requirements

- macOS
- Python 3
- Accessibility permission for Terminal or your Python interpreter

## Setup

```bash
python3 -m pip install -r requirements.txt
```

Then grant Accessibility access:

System Settings > Privacy & Security > Accessibility

Add Terminal or the Python interpreter you use to run Snappy.

## Run

```bash
python3 main.py
```
